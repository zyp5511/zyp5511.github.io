# REFLEX Project Page

This repository hosts the REFLEX paper website in a minimal Jekyll structure.

## Local preview

If Jekyll is installed:

```bash
bundle exec jekyll serve
```

or

```bash
jekyll serve
```

Then open `http://localhost:4000`.

## Structure

- `_config.yml`: Jekyll site config
- `_layouts/default.html`: shared HTML layout
- `index.html`: homepage content with front matter
- `assets/css/styles.css`: site styles
- `assets/fig/`: figure assets
